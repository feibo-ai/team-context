## D-04 · `plan_approve` gate 严格度

**Chose**: 软 gate · 警告 + 显式确认

**Implication**:
- MCP `plan_approve` 工具检测 issue 无 `plan-approved` label 时：
  - 打印警告："This issue has no plan-approved label. SOP v0.4 non-negotiable #1 requires a reviewed plan before implementation."
  - 要求用户回："yes-skip-sop" 字符串才执行
  - 不回直接拒
- 跑 1 个月统计绕过率
- 月度健康度报告里加 "Soft gate bypass count"
- 阈值：
  - 绕过率 < 5% → 维持软 gate
  - 5-30% → 月度复盘讨论原因
  - \> 30% → gate 设计本身有问题，重新评估

**How to apply**:
- Plan 2 Task 10 (plan_approve.ts) 加 zod schema field `confirmBypass: z.literal("yes-skip-sop").optional()`
- Plan 2 Task 15 (monthly_health_report.ts) 加 bypass count 统计逻辑
- 工具 description 里明确写"Soft gate per AI MIQ v0.2 D-04"
